Leetcode337.House Robber III

leetcode link

record in 2022/9/8

This problem is a combination of DFS and dynamic programming. The key to solving the problem is to realize that given a root, the maximum value we can get is by deciding to choose or not to choose it. If not to choose, then the current maximum would be the sum of both its child nodeâ€™s choosing value. If to choose, that means we can still decide whether to choose based on the comparison. We finish the dynamic process by searching the tree from the bottom to the top.

The code is quite simple and straghtforward but the idea is hard to come up with in the first time.
